# Chat Web App

هذا المشروع هو تطبيق محادثة ويب بسيط يحتوي على واجهة مستخدم تفاعلية. يتضمن المشروع الملفات التالية:

## الهيكل

- `src/index.html`: الملف الرئيسي لصفحة الويب، يحتوي على هيكل الصفحة ويستدعي ملفات CSS وJavaScript اللازمة.
- `src/components/chat-buttons.html`: يحتوي على الأزرار اللازمة لتطبيق المحادثة، يتم تضمينه في `index.html`.
- `src/styles/main.css`: يحتوي على أنماط CSS لتنسيق صفحة الويب، بما في ذلك الأزرار والعناصر الأخرى.
- `src/scripts/main.js`: يحتوي على الشيفرة البرمجية الخاصة بتطبيق المحادثة، مثل التعامل مع الأحداث المرتبطة بالأزرار.
- `package.json`: ملف التكوين لـ npm، يحدد التبعيات والبرامج النصية للمشروع.
- `.gitignore`: يحدد الملفات والمجلدات التي يجب تجاهلها من قبل نظام التحكم في الإصدارات Git.

## الإعداد

لتشغيل المشروع، تأكد من أن لديك Node.js مثبتًا. ثم يمكنك تثبيت التبعيات باستخدام الأمر التالي:

```
npm install
```

## الاستخدام

بعد تثبيت التبعيات، يمكنك فتح `src/index.html` في متصفحك لعرض التطبيق.
